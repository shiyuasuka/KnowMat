EXTRACTION INSTRUCTIONS FOR COMPOSITION FIELDS:
═══════════════════════════════════════════════════════════════════════════
For each composition entry, extract the following fields:

1. source_doi: Extract the paper's DOI from anywhere in the text (title area, header, footer, abstract).
   Format: '10.XXXX/...' (e.g., '10.1016/j.msea.2024.147225'). Use null if not found.

2. composition_normalized: Rewrite composition using ONLY element symbols and atomic percentages.
   Remove abbreviations, parenthetical codes, and sample names.

3. main_phase: Identify the primary crystal structure from XRD/EBSD/electron diffraction.
   Use standard abbreviations: BCC, FCC, HCP, amorphous, or combinations.

4. has_precipitates: true if ANY secondary phases or precipitates are mentioned.

5. grain_size_avg_um: Extract average grain size in micrometers.
   If reported in nm, convert to um (divide by 1000).

6. grain_size_text: Quote original grain size description with method/conditions.

7. process_category: One of AM_DED, AM_LPBF, AM_SLM, SPS, Arc_Melting, HeatTreat, Casting, Unknown.

8. processing_params: Structured key process parameters (JSON object) when explicit.
   Standard keys:
   Laser_Power_W, Scan_Speed_mm_s, Layer_Thickness_um, Hatch_Spacing_um,
   Preheat_Temperature_C, Shielding_Gas, Oxygen_Content_ppm, Build_Orientation.

9. build_orientation: Direction-specific marker when relevant (Parallel-BD, Perpendicular-BD, etc.).
   If one composition has data under multiple directions, create separate entries.

10. xrd_details: XRD instrument info and phase identification only.

11. microstructure_description: Morphology from SEM/EBSD/TEM.

12. measurement_condition for properties:
   ALWAYS start with test temperature in Kelvin when known.
   Format: 'at XXX K; [other conditions]'.
   If paper gives C, convert to K (C + 273).
   Room temperature = 298 K.

13. role:
   - Target: synthesized/processed/tested in this paper
   - Reference: only cited from other papers
   Default to Target when in doubt.
