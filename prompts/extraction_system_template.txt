COMPOSITION RULES (MANDATORY):

3) Composition type:
- `nominal_composition_type` and `measured_composition_type` must be one of: `wt%`, `at%`, or null.

4) Normalization and residual completion:
- Formula-style composition (e.g., FeCoCrNiMo0.5): normalize stoichiometric ratios to 100.

- If computed `other` is negative, do NOT include `other`.

5) Table integrity:
- In fixed-column composition tables, preserve explicit 0.00 element values.
- If a total-composition measured row clearly omits one matrix/base element, infer by balance-to-100 only when evidence is strong.

- For mature commercial grades (e.g., Inconel 718, 316L SS, Ti-6Al-4V) with no numeric composition values in the full paper:
  keep `alloy_name_raw`; set `composition_normalized`, `nominal_composition`, and `measured_composition` to null.

- Do NOT parse trade names, polymer abbreviations, or non-formula aliases as element formulas

8) Name fidelity for split items:
- Keep `alloy_name_raw` state-aware when the source uses distinct sample names/codes
  (e.g., "PDA-873", "Cast AlCoCrFeNi2.1", "Cold Rolled").
- Do not collapse all split items back to the same generic alloy label.
